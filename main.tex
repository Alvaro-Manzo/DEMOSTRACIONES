\subsection{Demostración de $1 = 0.999\ldots$}

Demostrar que:

\[
1 = 0.999\ldots
\]

Si dividimos:

\[
\frac{1}{3} = 0.333\ldots
\]

Y volvemos a multiplicar por 3:

\[
3 \cdot \frac{1}{3} = 0.999\ldots
\]

\[
 \frac{3}{3} = 0.999\ldots
\]
Simplificamos la fraccion:

\[
\frac{3}{3} = 1
\]

\[
\therefore 1 = 0.999\ldots
\]

\subsection{Demostración de $1>0$}

Demostrar que:

\[
1 > 0
\]

Para esta demostración utilizaremos el método de contradicción.

Existen tres posibilidades para comparar $1$ y $0$:

\[
1 = 0, \qquad 1 < 0, \qquad 1 > 0
\]

\textbf{Caso 1:}

Supongamos que

\[
1 = 0
\]

Usamos la propiedad aditiva de la igualdad:

\[
\textit{Si } a=b \textit{ entonces } a+c=b+c
\]

Sumamos $-1$ a ambos lados:

\[
1 + (-1) = 0 + (-1)
\]

Simplificando:

\[
0 = -1
\]

Esto es una contradicción.

\[
\therefore 1 \neq 0
\]

\textbf{Caso 2:}

Supongamos ahora que

\[
1 < 0
\]

Restamos $1$ en ambos lados:

\[
1-1 < 0-1
\]

Simplificando:

\[
0 < -1
\]

Lo cual es falso.

Por lo tanto, las dos primeras posibilidades son falsas y concluimos que:

\[
\therefore 1 > 0
\]
