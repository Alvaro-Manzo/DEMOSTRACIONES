\subsection{Demostración de $1 = 0.999\ldots$}

Demostrar que:

\[
1 = 0.999\ldots
\]

Si dividimos:

\[
\frac{1}{3} = 0.333\ldots
\]

Y volvemos a multiplicar por 3:

\[
3 \cdot \frac{1}{3} = 0.999\ldots
\]

\[
 \frac{3}{3} = 0.999\ldots
\]
Simplificamos la fraccion:

\[
\frac{3}{3} = 1
\]

\[
\therefore 1 = 0.999\ldots
\]

\subsection{Demostración de $1>0$}

Demostrar que:
\[
1 > 0
\]

Para esta demostración utilizaremos el método de contradicción.

En este caso existen tres posibilidades para comparar $1$ y $0$:

\[
1 = 0, \qquad 1 < 0, \qquad 1 > 0
\]

Para el primer caso usaremos contradicción:

\[
1 = 0
\]

Usaremos:
\[
1 \neq 0
\]

Por la propiedad aditiva de la igualdad:


\[\qquad\textit{Si a = b, entonces a + c = b + c}\]

\Longrightarrow

\[\text{Aplicamos la propiedad aditiva de la igualdad sumando
1 y -1 a ambos lados:}\]

\[
1 -1 = 0 + 1
\]

Simplificando obtendremos:

\[
0 = 1
\]

Llegando a una contradicción con nuestra premisa.

\[
\therefore \quad 1 \neq 0
\]

Sigamos con la siguiente situación:

\[
1 < 0
\]

\[\text{Al igual que la situacion anterior ocuparemos la propiedad aditiva de la igualdad:}\]

\[
\quad{1 - 1 < 0 - 1}
\]

\[\text{Simplificando:
}\]

\[
0 < -1 
\]

Lo cual es falso.

\[
\therefore\]

\textit{Ya que las primeras dos proposiciones resultaron ser falsas solo nos queda aceptar la proposición:}


\[
1 > 0
\]
